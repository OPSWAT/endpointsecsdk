# Compliance / Device Trust Artifacts

Compliance and Device Trust artifacts are organized by operating system.
These artifacts typically include:
- Platform-specific compliance libraries
- Integration notes
- Platform metadata (e.g., moby)
- Support charts (HTML/XML)

In the extracted package, these are commonly found under:

```
extract/analog/client/compliance/
```

## Typical Layout
- **bin/** – Platform-specific compliance libraries
- **docs/** – Integration notes, platform metadata, and support charts
